## 概要
このサイトは、私のスキルや経歴、ブログ記事などをまとめたポートフォリオサイトです。

## 使用技術
- Next.js
- TypeScript
- Tailwind CSS
- Vercel

## こだわった点
- **デザイン**: シンプルで見やすいデザインを心がけました。
- **パフォーマンス**: Vercelにデプロイし、高速な表示を実現しました。
- **レスポンシブ対応**: PCでもスマートフォンでも快適に閲覧できるように、レスポンシブデザインに対応しました。
